Arduino Lessen

Tijdens 4 sessies:

  • gebruik je eenvoudige sketches om de Arduino UNO experimenteerkit (geen tijd voor breadboarding!) aan te sturen
  • combineer je elementaire sketches tot een werkend geheel,
  • gebruiken je een final state machine als programmeerhulp,
  • leer je gestructureerd een nieuw Arduino project te starten.

Arduino Project

De 4e sessie concretiseer je de opgedane kennis met een origineel, zelfstandig project met (bij voorkeur) eigen hardware of de Arduino UNO experimenteerkit.
Na het volgen van de masterclass heb je voldoende kennis opgedaan om zelfstandig een eenvoudig Arduino project te realiseren.

Welke tools?

Je gebruikt de Arduino UNO experimenteerkit, de Arduino IDE en een finite state machine.

Voorkennis

  • De Arduino IDE heeft voor jou weinig geheimen.
  • Je begrijpt het belang van een korte cyclustijd.
  • Je weet hoe je millis() gebruikt om delay(n) te vervangen.
